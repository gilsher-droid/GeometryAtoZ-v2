# Fundamatics Learning Platform

## Product Vision

The Fundamatics Learning Platform is an interactive educational platform designed to help students think like mathematicians.

It is not a collection of lessons.

It is a learning engine.

Every course on the platform is built from reusable learning activities that encourage exploration, reasoning, construction, communication, and reflection.

The platform separates educational content from software behavior.

Teachers create lessons.

The engine delivers the learning experience.

Students build mathematical understanding by interacting with dynamic activities instead of passively consuming information.

The first course built on the platform is Geometry A to Z.

Future courses may include:

- Algebra
- Functions
- Probability
- Calculus
- Statistics
- Physics
- Economics

The long-term goal is to create a reusable educational platform that supports meaningful learning across many disciplines.
